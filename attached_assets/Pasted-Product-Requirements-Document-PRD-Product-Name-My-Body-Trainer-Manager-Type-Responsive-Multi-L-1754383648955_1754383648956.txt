Product Requirements Document (PRD)
Product Name: My Body Trainer Manager
Type: Responsive Multi-Lingual Web Application
Supported Languages: English, Spanish, French, Portuguese

________________________________________
1. Overview
Goal:
To create a responsive, multi-lingual web application that allows personal trainers to manage their clients, provide tailored training plans, monitor progress, and communicate directly. It includes client-side features for tracking workouts and inputting evaluations, and an overarching SuperAdmin system to manage trainers.
________________________________________
2. User Roles and Permissions
Role	Description	Key Access Points
SuperAdmin	Manages trainer accounts and system oversight	Admin dashboard, trainer account management
Trainer	Provides coaching, manages clients & content	Trainer dashboard, chat, plans, clients, reports
Client	Follows workout plans and tracks progress	Client dashboard, training plan, evaluations
